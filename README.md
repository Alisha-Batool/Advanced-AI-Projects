## 📰 News Topic Classification using BERT

### 📖 Overview

This project focuses on building a Natural Language Processing (NLP) model to classify news headlines into different categories using a transformer-based architecture.

### 🔍 Key Steps

✔ Loaded and explored the AG News dataset\
✔ Tokenized text using BERT tokenizer\
✔ Fine-tuned a pre-trained BERT model\
✔ Trained using Hugging Face Trainer API

### 🤖 Model Used

✔ BERT (bert-base-uncased)

### 📊 Evaluation Metrics

✔ Accuracy Score\
✔ F1-score

### 🎯 Objective

To classify news headlines accurately using advanced transformer-based NLP techniques.

## 🤖 Context-Aware Chatbot using RAG

### 📖 Overview

This project implements a chatbot that retrieves relevant information from a knowledge base before generating responses, making it more context-aware and intelligent.

### 🔍 Key Steps

✔ Created a custom knowledge base
✔ Split documents into smaller chunks
✔ Generated embeddings using sentence transformers
✔ Stored embeddings using FAISS
✔ Retrieved relevant context based on user queries

### 🤖 Techniques Used

✔ Retrieval-Augmented Generation (RAG)
✔ Vector similarity search

### 📊 Features

✔ Context-based responses
✔ Semantic search
✔ Interactive chatbot interface

### 🎯 Objective

To build a chatbot that provides accurate responses by retrieving relevant information instead of relying only on pre-trained knowledge.
